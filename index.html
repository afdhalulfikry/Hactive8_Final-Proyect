Interest

Jaringan komputer adalah sekelompok komputer yang terhubung satu sama lain untuk berbagi sumber daya dan informasi. Ada beberapa jenis jaringan komputer, salah satunya adalah arsitektur klien-server. Dalam tipe jaringan ini, simpul dapat berupa server atau klien. Simpul server memberikan sumber daya seperti memori, tenaga pemrosesan, atau data ke simpul klien. Simpul server juga dapat mengelola perilaku simpul klien. Klien dapat saling berkomunikasi satu sama lain, tetapi tidak dapat berbagi sumber daya.
